# Problem Statement
Grocery stores often face difficulties in managing items, updating stock, handling billing, and maintaining separate access for owners, cashiers, and customers. A simple and efficient console-based system is needed to automate these operations.

# Scope of the Project
The project covers inventory management, user authentication, billing calculations, stock updates, and basic customer account handling. It is designed for small retail stores and focuses on smooth data handling using Python.

# Target Users
- **Store Owners**
- **Cashiers**
- **Customers**

# High-Level Features
- Owner can add/remove items, edit prices, and update stock.
- Cashier can securely log in and verify customer bills.
- Customers can create accounts, log in, search items, and generate bills.
- Automatic bill calculation with item price and quantity.
- Structured menu-based navigation for ease of use.
