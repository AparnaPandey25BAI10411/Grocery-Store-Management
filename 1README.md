# Grocery Store Management System

## Overview
The Grocery Store Management System is a Python-based console application that helps manage daily store operations. It allows owners to manage inventory, cashiers to verify bills, and customers to create accounts, search items, and generate bills.

## Features
- Owner login with item add/remove/update options  
- Cashier login with bill verification  
- Customer account creation and login  
- Dynamic billing system  
- Inventory and price management  
- Secure authentication for each role  

## Technologies / Tools Used
- **Python 3**
- Dictionaries for data storage
- Console-based user interface

## Steps to Install & Run
1. Install Python 3 on your system.
2. Download or clone this project:
   ```
   git clone <repository-link>
   ```
3. Navigate to the project folder:
   ```
   cd grocery-store-management
   ```
4. Run the program:
   ```
   python main.py
   ```
